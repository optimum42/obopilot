# Installation

## Clone Repository

```bash
git clone <repository-url>
cd <project-folder>
```

---

## Create Virtual Environment

### macOS / Linux

```bash
python -m venv .venv
source .venv/bin/activate
```

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

---

## Install Project

```bash
pip install -e .
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```
